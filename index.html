Pokemon TCG for Gameboy Card Pack Simulator

Enter the info below and click submit to generate booster packs. Just a very simple demo, very hard to use if you're not the person who wrote the code. The first 3 text boxes can be found by using a debugger and going to those memory addresses when you're about to open the pack (such as on the PC pack selection screen). The number is how many packs you simulate. The final value, pack type, is a bit trickier. Set a debug breakpoint at 07:61C5 and input the value of 'a' when the breakpoint triggers. Alternatively, for PC packs, you can find the pack type from this list and then get the index for that pack type on this list. I promise I'm making this easier to use... Eventually.
